LLM provider

Tab Triage works with any provider you bring a key for. Your key stays on this device and is sent directly to the provider's API — no proxy in between.

Free LLM via OpenRouter

Don't want to pay any provider? OpenRouter exposes several models for free (rate-limited but generous for tab triage). Setup, top to bottom:

  1. Sign up at openrouter.ai — no credit card required for the free tier.
  2. Go to openrouter.ai/keys and create an API key. Copy it.
  3. In the fields above:
    • Provider: OpenAI-compatible
    • API key: paste the OpenRouter key
    • Base URL: https://openrouter.ai/api/v1
    • Model: pick one with the :free suffix from the free model list. Good defaults today: deepseek/deepseek-chat-v3-0324:free, google/gemini-2.0-flash-exp:free, meta-llama/llama-3.3-70b-instruct:free.
  4. Click Save, then Test connection above.

Free models have per-day request limits and can be slower or noisier than paid ones. If a triage fails, swap in a different :free model and retry. Your key and choice stay on this device.

Auto-triage

When you open new tabs, Tab Triage can quietly re-cluster the tabs in that window into native Chrome tab groups. This may move a tab into a different group if the LLM thinks it fits better — your manual groupings are not preserved. No tabs are ever closed. Pinned tabs are left alone.

Notion export

Send saved sessions and triage results straight into a Notion page. One-time setup:

  1. Create an internal integration at notion.so/my-integrations and copy its token (starts with ntn_ or secret_).
  2. Open the Notion page you want exports to live under. Click the ⋯ menu → Connections → invite your integration.
  3. Paste the token and the page URL below.

Sync

When enabled, your saved sessions are mirrored to Chrome's built-in sync (the same channel that syncs bookmarks and history). They'll appear on every browser you're signed into with this Google account. No data passes through any server we run.

Sessions larger than ~8 KB (rare; a 50-tab session is around 5 KB) stay local-only. Disabling sync also clears Tab Triage's keys from the sync silo.

Settings backup

Export your current settings to a JSON file, or restore from a previous export to set up a new profile quickly. The file includes your API keys — keep it private.

Privacy

  • Your API key is stored only in this browser's local storage.
  • Tab titles and URLs are sent to Anthropic when you click Triage. Nothing else leaves your machine.
  • No analytics, no tracking pixels, no third-party servers.
